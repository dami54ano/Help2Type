What is help2type?

Help2type is the first compact, haptic keyboard for smartphones. It can be connected to the smartphone via Bluetooth. It is flexibly attached to the smartphone with a watch band system and magnets. The keyboard can be folded onto the screen or simply stored under the device. It is a keyboard modeled after a laptop. Countless other symbols and special characters can be easily typed via function and command keys.

The well-known qwertz layout is printed. However, the software adapts to the selected language of the smartphone. The integrated software is optimized for 7 languages. For example, the qwerty or French azerty layout can be used.

The keyboard is compatible with Apple and Android devices.

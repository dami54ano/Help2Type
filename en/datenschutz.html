3. Data Collection on This Website

Cookies

This website uses cookies. Cookies are text files that are stored on your computer and enable an analysis of your use of the website. The information generated by cookies about your use of this website is generally not transmitted to a server.

Which cookies are used?

  • Necessary Cookies: These cookies are required to enable you to navigate through the pages and use essential functions.
  • Cookie Consent: We store your consent in a cookie so that you are not asked again on each visit.

Server Log Files

The page provider automatically collects and stores information in so-called server log files, which your browser automatically transmits to us. These are:

  • Browser type and version
  • Operating system used
  • Referrer URL
  • Hostname of the accessing computer
  • Time of the server request
  • IP address

This data is not merged with other data sources. The collection of this data is based on Art. 6 Para. 1 lit. f GDPR.

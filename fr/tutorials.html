Annonce nouveau logiciel

Dans ce tutoriel, vous en apprendrez plus sur nos nouvelles options linguistiques maintenant disponibles sur le clavier Help2type, ainsi que trois façons d'accéder au nouveau logiciel.

La première option vous permet de le faire confortablement depuis chez vous en suivant notre tutoriel dans la vidéo suivante. Deuxièmement, vous avez la possibilité de nous renvoyer le clavier pour que nous effectuions la mise à jour pour vous. Cette option peut être réservée via notre boutique en ligne. La troisième variante consiste à envoyer le clavier à l'un de nos distributeurs agréés qui effectuera la mise à jour pour vous.

Choix de langue

Dans ce tutoriel, vous apprendrez comment sélectionner la langue sur votre clavier. Pour cela, utilisez la combinaison de touches cmd, fn et le chiffre correspondant pour votre langue souhaitée. Les options disponibles sont les suivantes : 0 pour l'italien, 1 pour l'anglais, 2 pour l'espagnol, 3 pour le français, 4 pour l'allemand (Allemagne), 8 pour le suisse allemand et 9 pour le tchèque.

Configurer le clavier

Dans ce tutoriel, vous apprendrez comment configurer correctement le clavier. La base est la disposition QWERTZ connue. Pour que les symboles et caractères spéciaux soient corrects, vous devez régler le clavier matériel et logiciel sur le suisse allemand. Sélectionnez Réglages, Général, Clavier, Claviers et sélectionnez-y Allemand Suisse ou Français Suisse QWERTZ. Ouvrez maintenant par exemple Notes et appuyez en bas à gauche sur "clavier suivant". Sélectionnez Allemand Suisse.

Attribution des touches

Dans ce tutoriel, vous apprendrez l'attribution des touches. C'est la disposition qwertz connue des ordinateurs portables. Avec la touche de fonction combinée avec la barre d'espace, le clavier tactile apparaît ou disparaît sur votre smartphone sous iOS. Intéressant pour vous si vous souhaitez modifier la disposition ou passer aux emojis. Pour les appareils Android et Samsung, veuillez lire la section dans le mode d'emploi. Command combiné avec la barre d'espace fait clignoter l'arrière-plan du clavier pendant 5 secondes.

Instructions d'utilisation

Comment écrire ?

Le clavier est optimisé pour écrire avec les deux pouces. Le style d'écriture a été inspiré de la jeunesse d'aujourd'hui qui le fait intuitivement de cette manière. La taille de vos doigts ne joue aucun rôle avec ce style. Votre cerveau apprend automatiquement où se trouvent les touches après environ 30 séances d'apprentissage. D'autres formes d'écriture sont possibles.

Instructions :

Tenez les deux paumes vers le haut. Joignez vos mains jusqu'à ce que les petits doigts se touchent latéralement. Placez le smartphone avec l'écran vers le haut dans la main. La prise de charge doit être orientée vers la paume de la main. Placez les pouces sur le clavier. Vous pouvez déjà commencer à taper.
